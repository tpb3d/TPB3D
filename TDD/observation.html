Observation

All code here is OpenGL both rendered by the TPB3D engines and 3rd party tools.

1: Viewing

The player’s main view of the park while the game is running. All code here is OpenGL rendered with the main engines.

Builder-
Normal view frustum or primary camera.
Ride-
Tie the a view frustum to a ride and allow seat/car choice.
Guest-
Peeping Tom or Tie the a view frustum to a person and take control.
Snapshots-
Point the view frustum at a location and snapshot to bit image and dialog for a save.
Baloon and Flying Cams-
Tie the a view frustum to a balloon, peep, bird or plane.
